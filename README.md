# PyKirigami Anaconda Commands
First Time (delete existing 2026-VIPTester folder from desktop):
```bash
conda activate kirigami && cd desktop && git clone -b PKV1 https://github.com/issacpv/2026-VIPTester && cd 2026-VIPTester && python run_sim.py
```

Any Other Time:
```bash
conda activate kirigami && cd desktop/2026-VIPTester && python run_sim.py
```
Should take 5 min to show results
