# PyKirigami Anaconda Commands
First Time (delete existing 2026-VIPTester folder from desktop):
```bash
conda activate kirigami && cd desktop && git clone https://github.com/issacpv/2026-VIPTester && cd 2026-VIPTester && python -m auxetic_studio
```

Any Other Time:
```bash
conda activate kirigami && cd desktop/2026-VIPTester && python -m auxetic_studio
```
Should take 5 min to show results
